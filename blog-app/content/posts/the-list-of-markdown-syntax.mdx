## Syntax List

Markdown is a lightweight markup language for creating formatted text using a plain-text editor. Below is a comprehensive guide to Markdown syntax:

---

### Headings

Use `#` for headings. The number of `#` determines the level of the heading.

```
# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6
```

---

### Bold and Italic

Use `**` or `__` for bold text and `*` or `_` for italic text.

```
**Bold Text**
_Italic Text_
```

---

### Lists

#### Unordered Lists

Use `-`, `*`, or `+` for unordered lists.

```
- Item 1
- Item 2
  - Sub-item 1
  - Sub-item 2
```

#### Ordered Lists

Use numbers followed by a period for ordered lists.

```
1. Item 1
2. Item 2
   1. Sub-item 1
   2. Sub-item 2
```

---

### Links

Use `[text](URL)` for links.

```
[Visit Example](https://example.com)
```

---

### Images

Use `![alt text](image URL)` for images.

```
![Markdown Logo](https://example.com/markdown-logo.png)
```

---

### Blockquotes

Use `>` for blockquotes.

```
> This is a blockquote.
```

---

### Code Blocks

Use backticks (`) for inline code and triple backticks (```) for code blocks.

#### Inline Code

```
Use `console.log()` to print to the console.
```

#### Code Block

```
function greet() { console.log("Hello, World!"); }
```

---

### Horizontal Rules

Use `---`, `***`, or `___` for horizontal rules.

```
---
```

---

### Tables

Use pipes (`|`) and dashes (`-`) to create tables.

```
| Syntax      | Description |
| ----------- | ----------- |
| Header      | Title       |
| Paragraph   | Text        |
```

---

### Task Lists

Use `- [ ]` for unchecked tasks and `- [x]` for checked tasks.

```
- [x] Completed Task
- [ ] Incomplete Task
```

---

This guide covers the most commonly used Markdown syntax. Experiment with these features to create beautifully formatted content!
